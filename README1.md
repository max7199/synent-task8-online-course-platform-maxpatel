# Online Course Platform

## 📌 Project Overview

Online Course Platform is a full-stack web application developed using React, Node.js, Express, and MongoDB. The platform allows users to browse courses, register, log in, enroll in courses, and track their learning progress.

## 🚀 Features

### User Features
- User Registration & Login
- JWT Authentication
- Browse Available Courses
- View Course Details
- Enroll in Courses
- User Dashboard
- Track Learning Progress
- Responsive Design

### Admin Features
- Add New Courses
- Edit Existing Courses
- Delete Courses
- Manage Course Content
- Manage Users

### Course Features
- Course Listings
- Course Categories
- Instructor Information
- Course Pricing
- Course Descriptions
- Learning Modules

## 🛠️ Technologies Used

### Frontend
- React.js
- React Router DOM
- Axios
- Context API
- CSS

### Backend
- Node.js
- Express.js
- JWT Authentication
- Bcrypt.js

### Database
- MongoDB
- Mongoose
